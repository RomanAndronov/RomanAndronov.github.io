Associativity

AecImpl.java:

In this step we develop the existing method AecImpl.java:addOperator() further by introducing a new one:

private int addSamePrecOperator( AeItem newop, AeItem currop )

which handles same-precedence operators respecting their associativity.

Since this is the first time we have to handle syntactic errors we also implement a new method:

private void mkErr( String where, String what, AeItem optitem )


Observe that at this stage AEC can handle operators' precedence and associativity. OOEMs are next:
