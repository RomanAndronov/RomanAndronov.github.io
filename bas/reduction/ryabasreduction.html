Scope Reduction

- if a current problem can not be solved then there exists a simpler similar problem that can not be solved
- find that simpler problem
- repeat until a problem can be solved
- using the knowledge gained from the solution of the simplest problem find the solution to the original problem potentially solving all the intermediate problems
